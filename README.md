# 🌟 Astro Lumière - Plateforme de Prise de Rendez-vous Astrologie

Bienvenue sur **Astro Lumière**, une plateforme web magnifique et moderne pour prendre des rendez-vous de consultation en astrologie directement via WhatsApp.

## ✨ Caractéristiques

- 🎨 **Design Premium** : Couleurs élégantes (or, pourpre, bleu profond) inspirées de l'univers
- ⭐ **Animation Fluide** : Étoiles scintillantes et animations fluides
- 📱 **Responsive** : Adapté pour tous les appareils (mobile, tablette, desktop)
- 🔗 **Intégration WhatsApp** : Lien direct pour contacter via WhatsApp
- 💬 **Services Variés** : Thème astral, consultations, prédictions, compatibilité
- 🌙 **Témoignages** : Section avec avis de clients satisfaits

## 🔧 Personnalisation

### Modifier le numéro WhatsApp

Ouvrez `index.html` et remplacez le numéro dans le lien:
```html
href="https://wa.me/[VOTRE_NUMERO]"
```

### Modifier les tarifs et horaires

Trouvez la section `.details` et mettez à jour:
```html
<p><strong>Tarifs:</strong> À partir de 500 DA</p>
```
